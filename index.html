🔗 Sincronización en Vivo

Usa la misma configuración en PC y teléfono para sincronizar likes, comentarios y shares en tiempo real.

  1. Abre console.firebase.google.com
  2. Crea proyecto (nombre: "augusto-gamer")
  3. Añade app web (icono </>)
  4. Habilita Realtime Database → Crear → Modo prueba
  5. Ve a Settings → Tu app → "Config"
  6. Copia el objeto JSON completo
  7. Pega abajo y acepta ✅ { "apiKey": "AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx", "authDomain": "augusto-gamer.firebaseapp.com", "databaseURL": "https://augusto-gamer.firebaseio.com", "projectId": "augusto-gamer", "storageBucket": "augusto-gamer.appspot.com", "messagingSenderId": "123456789012345678", "appId": "1:123456789012:web:abcdef1234567890" }

En modo local los datos se guardan solo en este dispositivo. Con Firebase todos los dispositivos ven el mismo historial.
